Tervetuloa Trotteniin

Tarvitset GitHub Personal Access Tokenin datan tallennukseen Gistiin.

1. Mene github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Klikkaa Generate new token (classic)
3. Valitse scope: ☑ gist — ei muuta
4. Kopioi token ja liitä alle

Sovellus luo uuden yksityisen Gistin automaattisesti.
